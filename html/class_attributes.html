Construtor.

Lê toda a tabela de atributos do arquivo .class. Cada entrada tem a seguinte estrutura:

AttributeInfo {
u2 attributeNameIndex;
u4 attributeLength;
std::string name;
u1 info[attributeLength];
}

Portanto, a atribuição dos atributos é feita verificando o nome decodificado da da entrada de índice attributeNameIndex da Tabela de Constantes.

Parâmetros
fArquivo .class a ser lido.
attributesCountComprimento do vetor Attributes.
constantPoolTabela de Constantes já lida do arquivo .class.
