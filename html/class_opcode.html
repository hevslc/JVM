Descrição detalhada

Classe base para Opcodes. Ainda não está pronta para ser uma classe completa para instruções, apenas para opcodes.

NOTA: Caso seja necessário que um opcode acesse outros elementos externos (CpInfo por exemplo) é só adicionar um ponteiro para o elemento como atributo dessa classe, criar um atributo binário para indicar a dependência e uma função para fazer a injeção da dependência.
