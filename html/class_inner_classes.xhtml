Descrição detalhada

Se a Tabela de Constantes de uma classe ou interface C contiver pelo menos uma entrada Cpindo com a estrutura Class que representa uma classe ou interface que não seja membro de um pacote, deverá haver exatamente um atributo InnerClasses na tabela de atributos de a estrutura ClassFile para C.
