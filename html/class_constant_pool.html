Construtor auxiliar

Lê toda a tabela de constantes do arquivo .class armazenando as informações no objeto da classe cpinfo em seus respectivos atributos. Cada entrada tem a seguinte estrutura:

Cpinfo {
u1 tag;
u1 info[ ];
}

Portanto, a atribuição das constantes é feita verificando a tag, o qual define o tipo da informação em cp_info.

Parâmetros
fArquivo .class a ser lido.
constantPoolCountComprimento do vetor.

◆ getDouble()

double ConstantPool::getDouble ( u4  highBytes,
u4  lowBytes 
)

Decodifica o valor float dos bytes dados à função. Os bytes highBytes e highBytes são primeiro convertidos em long. Então:

  • Se bytes é 0x7ff0000000000000L, o valor double será infinito positivo.
  • Se bytes é 0xfff0000000000000L, o valor double será infinito negativo.
  • Se bytes no intervalo [0x7ff0000000000001L, 0x7fffffffffffffffL] ou no intervalo [0xfff0000000000001L, 0xffffffffffffffffL], então o valor double será NaN (not a number).
  • Em outros casos, seja s, e, e m três valores que são computados de bytes :
    1. int s = ((bytes >> 63) == 0) ? 1 : -1;
    2. int e = (int)((bytes >> 52) & 0x7ffL);
    3. long m = (e == 0) ? (bytes & 0xfffffffffffffL) << 1 : (bytes & 0xfffffffffffffL) | 0x10000000000000L; Então o valor numérico será resultado da expressão matemática .
      Parâmetros
      highBytesBytes mais significativos do valor numérico (Big-endian) de ponto flutuante IEEE 754.
      lowBytesBytes menos significativos do valor numérico (Big-endian) de ponto flutuante IEEE 754.
      Retorna
      valor double decodificado

◆ getFloat()

float ConstantPool::getFloat ( u4  bytes)

Decodifica o valor float dos bytes dados à função. A entrada bytes é do tipo int. Então:

  • Se bytes é 0x7f800000, o valor float será infinito positivo.
  • Se bytes é 0xff800000, o valor float será infinito negativo.
  • Se bytes no intervalo [0x7f800001, 0x7fffffff] ou no intervalo [0xff800001, 0xffffffff], então o valor float será NaN (not a number).
  • Em outros casos, seja s, e, e m três valores que são computados de bytes :
    1. int s = ((bytes >> 31) == 0) ? 1 : -1;
    2. int e = ((bytes >> 23) & 0xff);
    3. int m = (e == 0) ? (bytes & 0x7fffff) << 1 : (bytes & 0x7fffff) | 0x800000; Então o valor numérico será resultado da expressão matemática .
      Parâmetros
      bytesBytes do valor numérico
      Retorna
      valor float decodificado
