◆ r2()

u2 r2 ( std::ifstream &  f)

Método r2.

Método lê 2 bytes do arquivo na forma big-endian: Lê o primeiro byte atribuindo-o a uma variável de 16 bits e desloca-o para os primeiros 8 bits (tornando-o mais significativo). Lê o segundo byte armazenando como menos significativo. Ou seja, b = byte1byte2.

Parâmetros
fArquivo .class a ser lido.
Retorna
b 2 bytes lidos na ordem big-endian.
